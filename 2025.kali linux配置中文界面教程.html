Kali Linux 2025 中文完全配置指南

一、系统更新与中文包安装

1. 更新软件源

sudo apt update && sudo apt upgrade -y

2. 安装核心中文支持包

sudo apt install locales language-pack-zh-hans kde-l10n-zhcn

二、语言环境配置

1. 生成中文locale

sudo nano /etc/locale.gen
# 取消注释以下行
zh_CN.UTF-8 UTF-8
zh_CN.GBK GBK

生成语言环境：

sudo locale-gen

2. 设置默认语言

sudo update-locale LANG=zh_CN.UTF-8 LC_ALL=zh_CN.UTF-8
